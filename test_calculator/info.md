# Go Calculator

Este é um projeto simples de calculadora em Go para demonstrar o uso de funções básicas e testes unitários.

## Estrutura do Projeto
- `calculator.go`: Lógica das operações.
- `calculator_test.go`: Testes unitários.

## Pré-requisitos
- [Go](https://go.dev/dl/) instalado (versão 1.18 ou superior recomendada).

## Como Rodar os Testes

Para executar todos os testes deste pacote, abra o terminal na pasta do projeto e execute:

```bash
go test -v
